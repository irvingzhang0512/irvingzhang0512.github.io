3.1. 基本使用

  • try：运行可能会抛出异常的代码。
  • except：该关键字有以下多种功能
    • 捕获一个或多个特定异常
    • 捕获其他所有异常
  • else：当没有出现异常时执行
  • finally：不管如何都执行
  • 其他：
    • 抛出的异常只会被一个except捕获。
