📡 🔥 Skill/MCP 腰部热门（5 条）

phnx-labs/agents-cli

一个面向 CLI 编码智能体的统一工具链，允许开发者在自有订阅下并行运行多个 AI 编码代理，并集成浏览器驱动、安全密钥管理等能力，值得关注它试图成为“编码代理的操作系统”。

✍️ phnx-labs

要点

  • 多代理并行与自有订阅：支持同时运行 Claude、Gemini、Cursor 等多个编码代理，且使用用户自己的 API 订阅，避免第三方中转费用或限制。

NicolasPrimeau/artel

自托管的 MCP 服务器与多智能体协调层，为 AI 智能体集群提供共享记忆、任务管理、消息传递和会话交接能力，任何能发起 HTTP 调用的智能体均可参与协作。

✍️ NicolasPrimeau

要点

  • 多智能体协调基础设施：通过 MCP 协议实现智能体间的共享记忆、任务分发、消息传递和会话交接，解决多智能体协作中的状态同步与知识共享难题

Perseus-Computing-LLC/perseus-vault

Perseus Vault 是一个本地优先的 Rust 工具，为 AI Agent 提供持久化、确定性记忆，通过 SQLite+FTS5 混合检索和加密审计链，解决“Agent 靠猜、我们靠记”的核心痛点。

✍️ Perseus-Computing-LLC

要点

  • 记忆与审计一体：采用双时序历史记录和 AES-256-GCM 加密，配合加密链接的审计日志，确保 Agent 记忆可追溯、防篡改。

netdata/netdata

Netdata 是一个高性能、分布式的全栈可观测性平台，通过内置的机器学习与 MCP 协议支持，让任何规模的团队都能以最低成本实现 AI 驱动的实时监控与异常检测。

✍️ netdata

要点

  • MCP 协议原生支持*：项目标签中明确包含 mcp，意味着它可作为 MCP Server 直接为 AI Agent 提供强大的基础设施监控上下文，让 Agent 能实时查询、分析并响应系统状态。

Justkokosixnine/polymarket-mcp-server

将 Polymarket 预测市场集成到 AI Agent 的 MCP Server，让 LLM 能实时获取市场数据、下订单并执行交易策略。

✍️ Justkokosixnine

要点

  • 技术亮点：基于 Model Context Protocol (MCP) 构建，使 Claude 等 LLM 可直接与 Polymarket 区块链预测市场交互，支持市场数据查询、订单簿分析、实时价格监控及自动下单

📡 🎯 Skill/MCP 每日精选（5 条）

Signet-AI/signetai

SignetAI 是一个本地优先的 AI Agent 基础设施，为智能体提供可移植的身份、记忆与密钥管理，解决跨模型和跨运行环境的持久化状态难题。

✍️ Signet-AI

要点

  • 本地优先的 Agent 状态层：基于 SQLite 自托管，让 Agent 拥有持久化的身份、记忆和秘密，无需依赖云服务，契合“主权 AI”趋势。

openlegion-ai/openlegion

一个强调安全性的自主 AI Agent 框架，通过 Docker 沙箱隔离与多 Agent 编排，让用户以自然语言描述需求即可构建可执行人类级别任务的 AI 团队。

✍️ openlegion-ai

要点

  • 安全优先设计：每个 Agent 在独立 Docker 容器中运行，结合权限控制与审计日志，解决自主 Agent 执行任意代码时的安全风险。

Fergana-Labs/stash

Stash 是一个面向编码 Agent 的团队共享记忆系统，通过将非结构化知识转化为可复用的结构化技能，解决多 Agent 协作中的上下文断裂与经验浪费问题。

✍️ Fergana-Labs

要点

  • 技术亮点：采用“记忆-技能-任务”三层架构，Agent 将工作成果自动提炼为 Markdown 格式的技能文件，存储在共享知识库中，并通过语义搜索在后续任务中按需加载相关技能，实现持续学习与经验复用。

headroomlabs-ai/headroom

Headroom 是一个智能压缩工具，能在工具输出、日志、文件和 RAG 片段进入 LLM 前将其压缩 60-95%，同时保持回答质量不变，极大节省 Token 成本。

✍️ headroomlabs-ai

要点

  • 技术亮点：提供库、代理服务器和 MCP Server 三种集成方式，支持 OpenAI、Anthropic、LangChain 等主流框架，可无缝嵌入现有 Agent 工作流

diegosouzapw/OmniRoute

OmniRoute 是一个免费的统一 AI 网关，通过单一端点接入 231+ 模型提供商（含 50+ 免费），并集成令牌压缩、智能回退与主流编码代理，大幅降低 AI 开发成本与复杂度。

✍️ diegosouzapw

要点

  • 技术亮点：采用 RTK + Caveman 双层压缩技术，可节省 15%–95% 的令牌用量；内置智能自动回退机制，保障服务高可用。
