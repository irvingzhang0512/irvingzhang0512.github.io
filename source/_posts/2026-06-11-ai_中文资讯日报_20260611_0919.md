### [绕开时间反向传播！MIT挑战30年经典，RNN一步跨入并行时代](https://mp.weixin.qq.com/s?__biz=MzIwMTc4ODE0Mw==&mid=2247720844&idx=1&sn=9a856828dea93c38abcd932921194351)

{% note default %} MIT提出Supervised Memory Training (SMT)方法，利用Transformer教师模型生成记忆标签，使RNN训练摆脱BPTT的时间展开，实现并行训练并保持推理高效。 {% endnote %}

> 📡 来源：PaperWeekly · 2026-06-10 14:03

{% note success %} **要点**
- SMT将RNN训练拆分为记忆表征和记忆更新两步，用Transformer编码器-解码器教师模型为每个时间步生成最优记忆状态，使RNN训练转化为一步监督学习，信用链路长度降为O(1)

- 实验表明，SMT在合成任务、图像生成和语言建模上均优于BPTT训练的RNN，且通过DAgger Memory Training (DMT)微调可缓解滚动漂移问题，展示出更好的scaling特性（记忆压缩率随训练计算增加而提升）。
